# Инструкция по запуску программы PoputkaKGAMT на Android эмуляторе
Примечание: на данный момент основная проверка выполнена на *Android*. 
Для iOS сиситемы я *не устанавливал* эмулятор и не подключал БД. 

## О преокте
PoputkaKGAMT — мобильное приложение на C# и .NET MAUI разработанная в среде разработки Microsoft Visual Studio. 
Проект подключен к БД Firebase RealTime database через файл google-services.json (папка Platforms > Android) 

## Требования
Для запуска проекта нужны:
- Visual Studio 2022/2026 с поддержкой .NET MAUI, Либо установить .NET MAUI workload и удобнее работать в JetBrains Rider.
- .NET SDK
- Android Emulator

## Как запустить
1. Склонируйте репозиторий.
2. Выберите конфигурацию запуска Android.
3. Запустите приложение на эмуляторе.

## Структура проекта
- `Models` — модели данных
- `Services` — сервисы работы с данными из БД
- `ViewModel` — логика представления
- `Resources` — ресурсы приложения
